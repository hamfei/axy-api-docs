# 公共信用大数据简介(API)

全民普惠信用管理公司旗下的“全国公共信用大数据平台”秉承“给您最有用的数据”。依托于社会信用体系建设信息共享机制，汇集全国行政机关、司法机关、仲裁机构在履职过程中形成的行政处罚、行政强制、行政检查、司法执行、违法失信、抽查抽检等公共信用数据信息。依法针对企业、自然人、法定代表人做出的处罚决定或裁定，均具有法律约束力和法律强制力，在被处罚或裁定后，其生产经营、社会行为、经济活动均受到较大的限制或禁止，是评定其履约能力、履约意愿、信用水平、偿还能力的重要数据信息。包含但不限于税务处罚、市场监管处罚、司法拍卖、法院执行、公安追逃、股权查封、环保处罚、安全处罚、上市监管、仲裁信息、联合惩戒名单、涉金融严重失信名单等56个领域616个维度的全量标准化公共信用大数据。助力金融智能风控前沿技术的落地应用。
